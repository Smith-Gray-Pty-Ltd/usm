# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in USM, please report it privately:

- **Email**: security@smith-gray.com
- **GitHub**: Open a [private security advisory](https://github.com/Smith-Gray-Pty-Ltd/usm/security/advisories/new)

Please do not file public issues for security vulnerabilities. We will respond within 48 hours.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |
